## 🚀 Getting Started

### View locally

```bash
# 1. Clone the repo
git clone https://github.com/kathiharshithreddy/KATHI-HARSHITH-REDDY.git

# 2. Open the portfolio
cd KATHI-HARSHITH-REDDY
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

No build step, no dependencies — just open `index.html` in any modern browser.

## 🌐 Deploying to GitHub Pages

1. Go to your repository → **Settings → Pages**
2. Under **Source**, select **Deploy from a branch**
3. Choose **`main`** branch, **`/ (root)`** folder
4. Click **Save**

Your portfolio will be live at `https://<username>.github.io/<repo-name>/`

## ✏️ Customisation

- **Personal info**: Edit the text in `index.html` (name, bio, links, projects)
- **Colours**: Change CSS variables in the `:root` block at the top of `css/style.css`
- **Projects**: Update the `#projects` section in `index.html` with real GitHub links
- **Contact form**: Replace the `setTimeout` mock in `js/script.js` with a real service ([Formspree](https://formspree.io/), [EmailJS](https://www.emailjs.com/), etc.)
- **CV download**: Add your CV as `assets/cv.pdf` and update the `#downloadCV` button href
